The Mixins

Breakpoints

Short description on how to use the mixin.

Example:

.element {
  @include bp(480px,1024px) {
    color: black;
  }
}

CSS Output:

@media screen and (min-width: 480px) and (max-width: 1024px) {
  .element {
    color: black;
  }
}

Single Side Border Radius

Short description on how to use the mixin.

Example:

.element {
  @include border-top-radius(5px);
}

CSS Output:

.element {
  border-top-right-radius: 5px;
  border-top-left-radius: 5px;
}

Single Transform

Short description on how to use the mixin.

Example:

.element {
  @include rotate(45deg);
}

CSS Output:

.element {
  transform: rotate(45deg);
}

Box Emboss

Short description on how to use the mixin.

Example:

.element {
  @include box-emboss(0.5,0.75);
}

CSS Output:

.element {
  box-shadow: rgba(255, 255, 255, 0.5) 0 1px 0, inset rgba(0, 0, 0, 0.75) 0 1px 0;
}

Placeholder Color

Short description on how to use the mixin.

Example:

input {
  @include placeholder-color(#222222);
}

CSS Output:

input::placeholder {
  color: #222222;
}

Sizing

Short description on how to use the mixin.

Example:

.element {
  @include size(25px,100px);
  &:after {
    @include square(10px);
  }
}

CSS Output:

.element {
  width: 25px;
  height: 100px;
}
.element:after {
  width: 10px;
  height: 10px;
}

Hoverer

Short description on how to use the mixin.

Example:

.element {
  @include hoverer(color, #555555, #222222);
}

CSS Output:

.element {
  color: #555555;
}
.element:hover {
  color: #222222;
}

Responsive

Short description on how to use the mixin.

Example:

.element {
  @include responsive(background-color, #FFFFFF, #DDDDDD, #BBBBBB);
}

CSS Output:

.element {
  background-color: #FFFFFF;
}
@media only screen and (max-width: 64em) {
  .element {
    background-color: #DDDDDD;
  }
}
@media only screen and (max-width: 40em) {
  .element {
    background-color: #BBBBBB;
  }
}

Triangle

Short description on how to use the mixin.

Example:

.element {
  @include triangle($color:grey, $center:true);
}

CSS Output:

.element {
  content: '';
  width: 0;
  height: 0;
  border: 0.5rem solid transparent;
  border-left-color: grey;
  position: absolute;
  top: 50%;
  left: 100%;
  transform: translateY(-0.5rem);
}

Position

Short description on how to use the mixin.

Example:

.element {
  @include position(absolute,null,null,10px,15px);
}
.absolute-element {
  @include absolute(null,25px,25px);
}
.relative-element {
  @include relative(null,5px,5px);
}
.fixed-element {
  @include fixed(10px,50px);
}

CSS Output:

.element {
  position: absolute;
  bottom: 10px;
  left: 15px;
}
.absolute-element {
  position: absolute;
  right: 25px;
  bottom: 25px;
}
.relative-element {
  position: relative;
  right: 5px;
  bottom: 5px;
}
.fixed-element {
  position: fixed;
  top: 10px;
  right: 50px;
}

Placeholders

Clearfix

Short description on how to use the placeholder.

Example:

.element {
  @extend %clearfix;
}

CSS Output:

.element {
  *zoom: 1;
  &:before,
  &:after {
    display: table;
    content: "";
    line-height: 0;
  }
    &:after {
    clear: both;
  }
}

Hide Text

Short description on how to use the placeholder.

Example:

.element {
  @extend %hide-text;
}

CSS Output:

.element {
  font: 0/0 a;
  color: transparent;
  text-shadow: none;
}
